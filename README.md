# 🛰️ SkyGuard

> **SkyGuard** is a student-developed drone detection system designed to address privacy and safety concerns in Hong Kong’s emerging Low-Altitude Economy (LAE).  
> It combines computer vision, regulatory awareness, and user-friendly design to make drone monitoring accessible and responsible.

---

## 💡 Overview
**SkyGuard** focuses on developing an affordable, vision-based drone detection solution that supports responsible drone operations while protecting public privacy in urban areas.
